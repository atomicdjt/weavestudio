# Golden Path Specification

## Promise

Turn fragmented information into a structured, reusable deliverable workflow.

## Journey

1. **Choose template** — primary gallery or guided demo  
2. **Paste source** — Source material panel  
3. **Structure nodes** — Apply to Input / Split into nodes  
4. **Organize** — edit, connect, categorize, review  
5. **Generate deliverable** — template `outputStructure` composition  
6. **Preview & export** — MD / PDF / project JSON  
7. **Save & reopen** — named workspace autosave + import  

See `ACCEPTANCE-CRITERIA.md` and in-app Docs for operator detail.
