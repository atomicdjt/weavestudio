# Commercial Packaging Plan

## Package contents (when authorized to release)

- Source code on approved branch  
- README, setup, architecture, user guide  
- Template-authoring notes (`docs/product/TEMPLATE-SYSTEM-SPECIFICATION.md` to expand)  
- Privacy / local-processing explanation  
- Feature reality matrix + known limitations  
- Example / guided demo workspace  
- Test results (`npm test`, `npm run build`)  
- License placeholder until owner selects terms  
- Deterministic zip of source (exclude `node_modules`, `dist`) + checksum  

## Non-actions without owner authorization

- Push / merge to main  
- Publish npm or public demo claim beyond current host  
- Attach a final license  

## Portfolio narrative

Product promise → golden path → local-first honesty → template system → deliverable quality → testable core.
