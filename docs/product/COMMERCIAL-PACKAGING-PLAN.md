# Commercial Packaging Plan

## Package contents (when separately authorized)

- Source code from the approved branch
- README, setup instructions, architecture notes, and user guide
- Template-authoring notes (`docs/product/TEMPLATE-SYSTEM-SPECIFICATION.md`)
- Local-processing and privacy explanation
- Feature reality matrix and known limitations
- Example guided-demo workspace
- Verification results (`npm test`, `npm run lint`, `npm run typecheck`, `npm run build`)
- Proprietary All Rights Reserved ownership notice (`LICENSE.md`)
- Deterministic source archive that excludes `node_modules`, `dist`, environment files, and internal QA artifacts, plus a checksum

## Non-actions without owner authorization

- Push or merge into an authoritative branch
- Publish to npm or make public-hosting claims beyond the verified production host
- Grant commercial licensing or acquisition rights without a separate written agreement

## Ownership boundary

WeaveStudio source and associated intellectual property are proprietary. Private-repository access does not itself grant a license; commercial licensing or acquisition requires a separate written agreement with the copyright owner.

## Portfolio narrative

Product promise -> golden path -> local-first honesty -> template system -> deliverable quality -> testable core.
