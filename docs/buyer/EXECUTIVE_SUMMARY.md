# WeaveStudio Executive Summary

WeaveStudio is a local-first visual workflow product for consultants and agencies that turn unstructured discovery material into reviewable, reusable professional deliverables.

## Included value

- React/TypeScript workflow canvas with templates, validation, snapshots, exports, and browser-local persistence.
- Public-demo-ready guided workflow for an agency/client-proposal use case.
- Optional OpenAI and Gemini direct-BYOK assistance behind explicit per-request consent; no bundled secret or required AI service.
- Acquisition documentation, test evidence, release manifest, and deterministic source package.

## Honest boundaries

No customers, revenue, accounts, cloud sync, collaboration, or regulated-workflow certification are included. Browser storage is not encrypted or durable cloud storage. AI output is a human-reviewed draft and direct BYOK is not server-side secret storage.
