## Product

WeaveStudio is a local-first visual workflow canvas that helps consultants, agencies, researchers, documentation teams, and operations professionals turn fragmented source material into structured, reviewable deliverables.

Users can choose a workflow template, add notes or source material, structure and connect editable nodes, run a workflow-readiness validator, generate an editable deliverable, and export Markdown, PDF, or re-importable project JSON.

## Implemented capabilities

- React and TypeScript visual canvas built with `@xyflow/react`.
- Professional workflow templates and guided demonstration flows.
- Browser-local named workspaces, autosave, snapshots, undo, and redo.
- Workflow outline, minimap, auto-layout, and keyboard-safe editing.
- Structural validation for completeness, review gaps, and export readiness.
- Editable deliverable generation.
- Markdown, text-oriented PDF, and project JSON export.
- Full owned-data backup, validated restore, import-as-new, and scoped clearing.
- Optional direct-BYOK OpenAI and Gemini assistance with explicit confirmation for each request and human review before applying output.
- Automated unit, lint, type-check, build, browser, buyer-verification, and deterministic-package workflows.

## Verification

A buyer can clone the agreed source and run:

```bash
npm ci
npm run verify:buyer
```

The verification gate runs type checking, linting, unit tests, production build, browser tests, and deterministic package generation. The closing package identifies the exact commit and includes release evidence and checksums.
