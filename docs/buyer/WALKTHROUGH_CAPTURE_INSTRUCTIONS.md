# WeaveStudio Walkthrough Capture Instructions

Because Antigravity automation lacks native video capture capability, use these manual instructions to record the video walkthrough.

## Environment Preparation
1. Open a clean browser profile (e.g., Chrome Guest mode) to ensure no extensions or personal data are visible.
2. Ensure no desktop notifications, chat apps, or personal emails are running.
3. Set your display resolution to exactly 1920x1080 (1080p).
4. Do not play any copyrighted music or audio in the background.

## Capture Tool
1. Use OBS Studio or your OS's built-in screen recorder (e.g., Windows Snipping Tool record feature or macOS Cmd+Shift+5).
2. Set recording format to MP4, framerate to 30fps or 60fps.

## Execution
1. Navigate to `https://weavestudio-demo.vercel.app/`.
2. Press Record.
3. Follow the sequence exactly as written in `WALKTHROUGH_SHOT_LIST.md`.
4. Keep cursor movements smooth and deliberate. Do not flick the mouse.
5. If you make a mistake, stop, discard the recording, and start over to ensure a continuous, polished take.

## Post-Processing
1. Add captions matching the `WALKTHROUGH_NARRATION.txt`.
2. Export as a 1080p MP4 file named `WeaveStudio-Walkthrough.mp4`.
3. Keep the file size under 50MB if possible for easy sharing.
