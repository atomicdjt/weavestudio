# Buyer Value Proof

## Clear product wedge

WeaveStudio is a local-first client-deliverable workflow studio for consultants and agencies turning fragmented discovery material into structured, reviewable outputs.

## Shipped end-to-end examples

| Starting point | Workflow | Result |
|---|---|---|
| Discovery notes | Source → structured nodes → review → export | Proposal-ready brief |
| Meeting transcript | Source → actions/decisions → validation → export | Action-plan draft |
| Research fragments | Source → evidence/risks → review → export | Executive-brief draft |

Each path is deterministic until an optional BYOK AI request is explicitly confirmed. AI output remains a draft for human review.

## Evidence boundaries

This repository contains source, tests, packaging, and product documentation. It does not include customer lists, revenue, verified market traction, regulated certifications, or a claim of zero security risk.
