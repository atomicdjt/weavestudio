# Acquisition Listing

## Title

WeaveStudio: Local-First Visual Workflow Canvas

## Short Description

Turn messy inputs into structured, exportable deliverables with a local-first visual workflow canvas.

## Long Description

WeaveStudio is a React-based browser app for mapping repeatable workflows. Users can organize notes, transcripts, requirements, incident logs, or SOP drafts into connected Input, Transform, Decision, Review, optional AI Assist Blueprint, and Output nodes. The app then creates deterministic previews and exports based on the current workflow content.

The current release is intentionally local-first. It runs as a static frontend, stores work in browser localStorage, and does not require a backend, account system, database, cloud sync, or external API.

The source/IP acquisition page is available at `/acquire`; `/buyer` redirects there for backwards compatibility.

- Frontend source code.

- Indie developers building local-first productivity tools.
- Operations consultants packaging repeatable client workflows.
- Documentation consultants and technical writers.
- Template creators who want software around their methods.

## Monetization Paths

- Desktop app with one-time licenses.
- Paid template packs for specific professional workflows.
- Consulting enablement tool for repeatable client deliverables.
- BYOK-ready AI Assist provider adapters as optional buyer-led extensions.
- Source licensing for adjacent local-first workflow products.

## Suggested Acquisition Range

$5,000 to $15,000 depending on support, exclusivity, branding transfer, and any included deployment or domain assets.

## Limitations

- Browser localStorage is not a substitute for durable cloud or filesystem storage.
- No collaboration, authentication, or cloud sync in this release.
- Generation is deterministic and assistive; it does not verify facts or rewrite content through AI.
- AI Assist is a blueprint node only; no API keys or live provider calls are included.
- Not suitable for regulated sensitive-work claims without additional product, legal, privacy, and security review.

## Handoff Terms

Upon purchase, the buyer can receive the source repository, current documentation, and any separately agreed branding or hosting assets. Post-sale support should be negotiated separately.
